## 🚀 Quick Start

### 1. Install Dependencies (Already Done ✓)
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```
Visit: `http://localhost:3000`

### 3. Customize Your Portfolio
Open the files mentioned in `PLACEHOLDERS.md` and replace all `[Placeholders]` with your information.

### 4. Add Your Profile Picture
Replace `/public/profile.jpg` with your professional photo (400x400px minimum).

### 5. Deploy
Choose one deploymentption:
- **Vercel** (Recommended): `vercel deploy`
- **Netlify**: Connect your GitHub repo
- **GitHub Pages**: Build and push
